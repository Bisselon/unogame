# FAST UNO GAME

## Installation du jeu
Fichier .env
```sh
DB_HOST=localhost
DB_USER=postgres
DB_PASSWORD=root
DB_NAME=fast_uno
```

### MacOS & Linux

```sh
python3 -m venv venv
```

```sh
source venv/bin/activate
```

### Windows

```sh
python3 -m venv venv
```

CMD

```sh
venv\Scripts\activate
```

Powershell

```sh
venv\Scripts\Activate.ps1
```

#
```sh
pip install -r requirements.txt
```

### Lancement du jeu

```sh
python main.py
```
